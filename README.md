# Yes or No App

A tiny static web app with two buttons:

- Click **Yes** to show a green screen.
- Click **No** to show a red screen.
- Click **Reset** to return to the default view.

## Run locally

Because this is a plain HTML/CSS/JavaScript app, you can open `index.html` directly in a browser or serve it with a simple static server:

```bash
python3 -m http.server 8000
```

Then open <http://localhost:8000>.
